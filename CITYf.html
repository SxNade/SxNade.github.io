#1 GAINING INITIAL FOOTHOLD


(a) Starting with a nmap Scan to Enumerate For ports and services running on Them

More on NMAP: Read More Here

{Below is a typical nmap command that enumerates the target services and ports to maximum!}

$ nmap -PA -sC -sV -p- 192.168.1.1

What does this nmap command do?

  • -PA → Tells nmap to use TCP/ACK For Host Discovery
  • -sC → Turns on script scanning
  • -sV → Turns on service Detection
  • -p- → Tells nmap to scan For all ports from 1-65535
  • IP{192.168.1.1} -> The IP address Of your Target

    • This will Return with a number of ports and services running on those ports
